# TimeGrid - タイムスケジュール管理アプリ

TimeGridは、リアルタイムで共同編集が可能なタイムスケジュール管理ウェブアプリケーションです。
複数の場所（ステージや会議室など）にわたる予定をグリッド形式で視覚的に管理できます。

## 主な機能

- **グリッドベースの編集**: 時間と場所の軸を持つグリッド上で予定を直感的に配置。
- **ドラッグ＆ドロップ**: 予定の移動や時間の伸縮がマウス操作で可能。
- **リアルタイム同期**: Server-Sent Events (SSE) を使用し、複数のユーザーが同時に編集しても変更が即座に反映されます。
- **場所のカスタマイズ**: 予定を追加する場所（列）を自由に追加・並び替え可能。
- **エクスポート・印刷**: 作成したスケジュールをJSON形式で保存したり、きれいに印刷したりできます。

## セットアップと実行方法

### 必須環境
- Node.js (v14以上推奨)

### インストール
リポジトリをクローンした後、依存関係をインストールします。

```bash
npm install
```

### 実行
サーバーを起動します。

```bash
npm start
```

ブラウザで `http://localhost:3000` にアクセスして利用を開始してください。

## 技術スタック

- **フロントエンド**: Vanilla JavaScript, CSS3, HTML5
- **バックエンド**: Node.js, Express
- **データベース**: SQLite3 (better-sqlite3)
- **同期**: Server-Sent Events (SSE)
